LC 42. Trapping Rain Water

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

Example 1:


Input: height = [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6
Explanation: The above elevation map (black section) is represented by array [0,1,0,2,1,0,1,3,2,1,2,1]. In this case, 6 units of rain water (blue section) are being trapped.

Example 2:

Input: height = [4,2,0,3,2,5]
Output: 9

Approach:
1. Brute Force (O(nÂ²), not interview optimal)

    For each index, find the max height to the left and max height to the right.

    Water trapped at index i = min(max_left, max_right) - height[i].

    Sum across all i.

2. Dynamic Programming Precomputation (O(n), O(n) space)

    Instead of recomputing max left/right for each index, precompute arrays:

    left_max[i] = highest bar from 0..i.

    right_max[i] = highest bar from i..n-1.
    
    Then:

        water[i] = min(left_max[i], right_max[i]) - height[i]

3. Two-Pointer Approach (O(n), O(1) space)

    Maintain two pointers: left=0, right=n-1.

    Maintain left_max, right_max.

    While left < right:

        If height[left] < height[right]:

            If height[left] >= left_max: update left_max.

            Else: water trapped += left_max - height[left].

            Move left++.

        Else:

            If height[right] >= right_max: update right_max.

            Else: water trapped += right_max - height[right].

            Move right--.


Code:

def trap(height):
    n = len(height)
    if n == 0:
        return 0
    
    left, right = 0, n - 1
    left_max, right_max = 0, 0
    water = 0
    
    while left < right:
        if height[left] < height[right]:
            if height[left] >= left_max:
                left_max = height[left]
            else:
                water += left_max - height[left]
            left += 1
        else:
            if height[right] >= right_max:
                right_max = height[right]
            else:
                water += right_max - height[right]
            right -= 1
    
    return water

# Example
print(trap([0,1,0,2,1,0,1,3,2,1,2,1]))  # Output: 6
